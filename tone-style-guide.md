# DJ Taye / Flatlyne Entertainment — Tone + Style Guide

Voice: confident, warm, service-first, and energetic.
Say it like a professional who knows how to move a room without making the event about himself.

Core message:
More than music — it’s an experience.

Use:
- clear event language
- warm confidence
- simple next steps
- proof from real setup and real service
- “your event / your crowd / your vision”

Avoid:
- sounding like a club flyer only
- overpromising
- vague hype
- too many calls-to-action
- “cheap,” “budget,” or pressure language

CTA library:
- Check Availability
- Plan My Event
- Text My Event Details
- Start My Booking Request
- Let’s Build the Soundtrack
- Send Event Details

Booking reply tone:
“Thanks for reaching out. Send me the event date, city, event type, and hours needed, and I’ll confirm availability from there.”
